# Nguyễn Tấn Đạt - ES6 - FCP

ES6 and Beyond Technical Challenge
----------

Pre-requisites: Node.js 4.x or later

**Install Project Dependences:** ` npm install`

**Run:** 
  
-  **build:** `npm run build`
-  **run server**: `npm run serve`
-  **test**: `npm run test`
-  **test-watch**: `npm run test-watch`
